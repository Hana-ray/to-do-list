# Smart To-Do List App

## Overview
This is a smart to-do list application built with HTML, CSS, and JavaScript. It helps users manage daily tasks through a clean and interactive interface. The app allows users to add tasks, set due dates, assign priority levels, mark tasks as completed, edit tasks, delete tasks, and store task data using localStorage.

## Features
- Add new tasks
- Edit existing tasks
- Mark tasks as completed or undo them
- Delete tasks
- Set due dates
- Show overdue tasks
- Assign priority levels (Low, Medium, High)
- Filter tasks by All, Pending, and Completed
- View total, completed, and pending task counts
- Clear all completed tasks
- Save tasks using localStorage

## Technologies Used

- CSS
- JavaScript
